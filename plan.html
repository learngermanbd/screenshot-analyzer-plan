Section 02

System Architecture

Three-stage pipeline: Analyze → Build → Export. Two backend services (Next.js + Python ML), shared via JSON.

Stage 1: Analysis (Konva Canvas)

  • Upload screenshot → AI detects elements
  • Measurement overlays on image
  • Color palette, font specs, spacing extraction
  • Output: structured JSON of all detected elements

Stage 2: Design Builder (Craft.js Canvas)

  • Auto-populate from AI detection OR manual build
  • Grid mode (quick layout) + Free-form mode (pixel-perfect)
  • Component library with categories & search
  • Property editor for all element styles
  • Multi-screen support with navigation flow

Stage 3: Prototype & Inspect

  • Prototype mode: click interactions, transitions, phone preview
  • Inspect mode: CSS, React, Android, iOS code for any element
  • Layer tree: view hierarchy, reorder, lock, hide
  • Share public prototype URLs

Stage 4: Code Export

  • React + Tailwind (Gemini API)
  • Vue + Tailwind (Gemini API)
  • HTML + CSS (client-side)
  • Jetpack Compose (Gemini API)
  • Kotlin/XML Layout (Gemini API)
  • JSON design file + PNG/SVG (client-side)
  • Live preview in Sandpack iframe

Section 04

Design Builder — Drag & Drop Canvas

Craft.js-powered canvas with dual modes. Auto-populated from AI analysis or built manually from component library.

Canvas Modes

Grid Mode (Quick Prototyping)

  • Elements snap to flex/grid containers
  • Responsive by default
  • Stack-based layout (rows, columns, grids)
  • Best for: quick wireframes, content-heavy screens

Free-form Mode (Pixel-Perfect)

  • Absolute positioning anywhere on canvas
  • Resize handles on all elements
  • Snap-to-grid guide lines
  • Best for: precise mobile UI design, matching screenshots

Component Library

Mobile Components

  • Button (primary, secondary, outline)
  • Text Input / Search Bar
  • Card / List Item
  • Navigation Bar / Tab Bar
  • Modal / Bottom Sheet
  • Toggle / Checkbox / Radio
  • Slider / Progress Bar
  • Avatar / Badge / Chip

Layout

  • Row / Column / Grid
  • Container / Spacer
  • Scroll View
  • Safe Area
  • Divider

Text & Media

  • Title / Heading
  • Body Text / Caption
  • Icon (built-in set)
  • Image Placeholder
  • Map Placeholder

From Screenshot (AI)

  • Auto-detected buttons with labels
  • Auto-detected input fields
  • Auto-detected text blocks
  • Auto-detected images/icons
  • Auto-detected navigation bars

Section 10

AI Vision Strategy

Primary: Gemini 2.0 Flash

1,500 free req/day. Native JSON output. Used for: screenshot analysis, code export (React, Vue, Android), hierarchy detection.

Fallback: Ollama + Qwen2.5-VL

Unlimited, local, private. For offline use or when Gemini quota is exhausted.

Detection: Florence-2

Purpose-built bounding box detection. Runs locally. MIT licensed.

API Optimization: Each analysis = 1 Gemini call. Each code export = 1 call. Cache exports by design hash. HTML/CSS export is client-side (0 calls). Max ~3,000 analyses+exports/day.
